## 📖 Overview

**Booru Tag Editor Pro** is a feature-rich prompt editor designed for AI artists and prompt engineers. It simplifies the process of building high-quality prompts for image generation models by providing an intuitive interface for browsing, organizing, searching, and combining thousands of Booru tags.

Whether you're creating prompts for **ComfyUI**, **Stable Diffusion**, **SDXL**, **Flux**, or other AI image generation tools, Booru Tag Editor Pro helps you build prompts faster while keeping your tag libraries organized.

# 🚀 Installation

## Requirements

* Python 3.10+
* Windows 10/11

---

## Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/booru_tag_editor_pro.git

cd booru_tag_editor_pro
```

---

## Create a Virtual Environment

Windows

```bash
python -m venv .venv

.venv\Scripts\activate
```

Linux/macOS

```bash
python3 -m venv .venv

source .venv/bin/activate
```

---

## Install Dependencies

```bash
pip install -r requirements.txt
```

---

## Run

```bash
python main.py
```
